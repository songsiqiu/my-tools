# 多功能工具箱 v2.0

一个集成了多种实用功能的桌面工具，开箱即用，无需Python环境。

## 功能特性

### 📝 标签生成器
- 直接在文本框输入标签内容
- 自动计算最优字体大小
- 支持中文字体
- 每组标签用空行分隔

### 🖼️ 图片裁剪转PDF
- 自动裁剪图片空白区域
- 支持多种图片格式：jpg, png, bmp, tiff, gif
- 两种处理模式：
  - 分别转换：每张图片生成单独的PDF
  - 合并为一个PDF：多张图片合并成一个文件
- 支持选择文件夹或多个文件

### 📄 PDF空白裁剪
- 裁剪PDF每一页的空白边缘
- 支持递归处理子文件夹
- 保持原有目录结构

## 使用方法

1. 双击运行 `多功能工具箱.exe`
2. 选择需要的功能标签页
3. 按照界面提示操作
4. 点击"打开输出文件夹/文件"按钮查看结果

## 系统要求

- Windows 64位系统
- 无需安装Python或其他依赖

## 技术栈

- Python 3.12
- Tkinter (GUI)
- Pillow (图像处理)
- PyMuPDF (PDF处理)
- ReportLab (PDF生成)
- PyInstaller (打包)

## 开发

### 环境配置

```bash
pip install Pillow PyMuPDF reportlab pyinstaller
```

### 运行源码

```bash
python tools_gui.py
```

### 打包为exe

```bash
pyinstaller 多功能工具箱.spec --clean
```

生成的exe文件位于 `dist/多功能工具箱.exe`
